# elm-svgbob

Fork of the great ASCII to SVG converter SvgBob by Ivan Ceras. See the example
in [`examples/Main.elm`](examples/Main.elm) it covers some more detailed
information.

```elm
"""
+------+   +-----+   +-----+   +-----+
|      |   |     |   |     |   |     |
| Foo  +-->| Bar +---+ Baz |<--+ Moo |
|      |   |     |   |     |   |     |
+------+   +-----+   +--+--+   +-----+
              ^         |
              |         V
.-------------+-----------------------.
| Hello here and there and everywhere |
'-------------------------------------'
"""
```
